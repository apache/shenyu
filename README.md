### 这是一个异步的,高性能的,跨语言的,反应式的API网关。我希望能够有一样东西像灵魂一样，保护您的微服务。参考了Kong，Spring-Cloud-Gateway等优秀的网关后，站在巨人的肩膀上，Soul由此诞生！

# Features

   * 支持各种语言,无缝集成Dubbo,SpringCloud。
   
   * 丰富的插件支持，鉴权，限流，熔断，防火墙等等。
   
   * 网关多种规则动态配置，支持各种策略配置。
   
   * 插件热插拔,易扩展。
   
   * 支持集群部署，支持A/B Test。
